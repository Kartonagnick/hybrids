#66-dev-util
============
В этой задаче нужно изготовить функцию `unquote`:  
Функция принимает на входе строку, и убирает обрамляющие кавычки  

```vbs
function unquote(str)
  dim quot: quot = """"
  if left(str, 1) & right(str, 1) = quot & quot then 
    unquote = mid(str, 2, len(str) - 2)
  else
    unquote = str
  end if
end function
```

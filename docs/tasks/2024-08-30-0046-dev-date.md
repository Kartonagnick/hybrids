#46-dev-date
============

Доработка инструмента `date`:  
  - [x] `value`  содержит класс `DTSValue`  
  - [x] `import` содержит класс `DTSImport`  
  - [ ] `export` содержит класс `DTSExport`  
  - [ ] `stamp`  
  - [ ] `range`  
  - [ ] `duration`  

Схема зависимостей:  

```
                +-------+            
            .---| value |---.
           |    +-------+    |
           |                 |
      +--------+        +--------+   +----------+  +-------+
      | import |        | export |   | duration |  | range |
      +--------+        +--------+   +----------+  +-------+
           |                 |
           |    +-------+    |
            `---| stamp |---'
                +-------+   
```

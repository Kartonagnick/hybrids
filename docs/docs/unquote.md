[![P]][M] unquote v0.0.1
==========================
Инструмент служит для удаления обрамляющих кавычек  
Например:  

```vbs

  dim text: text = chr(34) & "text" & chr(34)


  WScript.Echo unquote("text")   ' text
  WScript.Echo twoDigits(text)   ' text
```

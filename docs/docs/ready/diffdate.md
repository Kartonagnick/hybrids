[![P]][M] ready/diffdate v0.0.1
===============================
Гибрид предназначен для получения длительности  
периода между двумя датами  
<br/>

Пример использования
--------------------
Открываем файл гибрида  `ready/diffdate.bat` на редактирование,  
и задаем значения:  
```
::'set "eBEG_DATE=2026-01-28 10:00:00"
::'set "eEND_DATE=2026-01-28 12:01:02"
```

Запускаем гибрид на выполнение.  
Ожидаемый результат: `2 hours, 1 min, 2 sec`
<br/>

Управляющие параметры
---------------------

| переменная | назначение                |
|:----------:|:-------------------------:|
| eDEBUG     | вкл/выкл отладочный вывод |
| eBEG_DATE  | начальная дата-время      |
| eEND_DATE  | конечная дата-время       |

```bat
set "eDEBUG=ON"
set "eBEG_DATE=2026-01-28 10:00:00"
set "eEND_DATE=2026-01-28 12:01:02"
```

[![P]][M] dir/delete v0.0.1
===========================
Процедура удаляет каталог вместе со всем содержимым:  
  - удаляет все подкаталоги каталога  
  - удаляет все файлы каталога  
  - удаляет сам каталог  

Прототип:  
```
sub deleteD(dir)
```

Пример использования:  

```vbs
deleteD "..\sandbox"
```
